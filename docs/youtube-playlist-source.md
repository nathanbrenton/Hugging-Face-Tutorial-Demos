Local offline assets are video files downloaded from the playlist.

Project convention:

- Playlist numbers use three digits: 001, 002, 003
- Demo folders use snake_case: video_002_pipeline_function
- Frontend values use kebab-case: video-002-pipeline-function
- Backend routes use kebab-case: /api/video-002/pipeline-function
- Not every playlist video requires a demo page
- Notes-only videos can be documented without a frontend/backend implementation
