002 - The pipeline function

This video introduces the Hugging Face pipeline() function. A pipeline provides a high-level interface that handles preprocessing, model inference, and postprocessing for common NLP tasks.

Sentiment Analysis

Pipeline call
pipeline("sentiment-analysis")
Model used
distilbert/distilbert-base-uncased-finetuned-sst-2-english
When the model runs
When the user clicks Analyze Text, the browser sends text to the local FastAPI backend. The backend runs the local Hugging Face model and returns structured JSON results.
